# InputField

Enhanced input field component with validation

## Real Usage Examples

### Usage 1

```tsx
<InputField
label="Full Name"
value={formData.name}
onChange={(value) => handleInputChange('name', value)}
required
placeholder="John Doe"
/>
```

### Usage 2

```tsx
<InputField
label="Email Address"
type="email"
value={formData.email}
onChange={(value) => handleInputChange('email', value)}
required
placeholder="john@example.com"
/>
```

### Usage 3

```tsx
<InputField
label="Company"
value={formData.company}
onChange={(value) => handleInputChange('company', value)}
placeholder="Acme Corp (optional)"
/>
```

## Props

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `label` | `string` | No | - | The input label |
| `type` | `enum` | No | `text` | The input type |
| `placeholder` | `string` | No | - | Placeholder text |
| `value` | `string` | Yes | - | Current input value |
| `onChange` | `(value: string) => void` | Yes | - | Change handler |
| `required` | `boolean` | No | `false` | Whether the input is required |
| `className` | `string` | No | `{"value":""}` | Additional CSS classes |
