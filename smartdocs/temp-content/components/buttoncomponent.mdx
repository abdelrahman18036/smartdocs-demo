# ButtonComponent

Advanced button component with state management for user interactions

## Real Usage Examples

### Usage 1

```tsx
<ButtonComponent variant="primary" size="small" className="w-full">
View Profile
</ButtonComponent>
```

### Usage 2

```tsx
<ButtonComponent
variant="primary"
onClick={() => window.location.href = '/contact'}
>
Contact Us
</ButtonComponent>
```

### Usage 3

```tsx
<ButtonComponent
variant="secondary"
onClick={() => addNotification({ message: 'GitHub repository opening...', type: 'info' })}
>
View on GitHub
</ButtonComponent>
```

## Props

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `children` | `ReactNode` | Yes | - | The content to display inside the button |
| `variant` | `enum` | No | `primary` | The visual style variant of the button |
| `size` | `enum` | No | `medium` | The size of the button |
| `disabled` | `boolean` | No | `false` | Whether the button is disabled |
| `onClick` | `() => void` | No | - | Click handler function |
| `className` | `string` | No | `{"value":""}` | Additional CSS class names |
