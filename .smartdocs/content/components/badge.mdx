# Badge

Badge Component

A versatile badge component for displaying labels, status indicators, and tags.
Supports multiple variants and sizes with customizable styling.

## Real Usage Examples

### Usage 1

```tsx
<Badge variant="primary" size="small">
{tech.category}
</Badge>
```

### Usage 2

```tsx
<Badge key={skillIndex} variant="secondary" size="small" className="mr-2">
{skill}
</Badge>
```

### Usage 3

```tsx
<Badge variant="primary" size="large">ID: {id}</Badge>
```

## Props

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `variant` | `enum` | No | `primary` |  |
| `size` | `enum` | No | `medium` |  |
| `className` | `string` | No | `{"value":""}` |  |
